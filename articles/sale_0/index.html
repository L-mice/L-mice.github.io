Sennheiser Momentum True Wireless
日本より海外のほうがはるかに安くなっていた。

はじめまして、かずまちといいます。17歳JK目指してます。 今回は初回ということで、ボクの好きなアイテムの紹介をしたいと思います。

今回紹介するアイテムというのは、ボクが日常的に使っている今となっては日用品とまでいえるものです。
タイトルでわかってしまった人もいるかもしれません。ワイヤレスイヤホンです。

Sennheiserという有名メーカーのもので、他の某メーカーとは違い、この会社が販売している唯一のワイヤレスイヤホンです。
ボクは、当時よく見ていたYouTuberが紹介していたのを見て、思わず買ってしまいました。 値段としては3万5000円ほどで決して安いものではありませんでしたが、
音質が良いことと見た目のかっこよさから、ほぼ迷うことなくこちらを購入してしまいました。

Amazonで届き、ワクワクして開封しました。 商品は問題なくボクの元に届いたので、すぐに音楽を聞いてみました。 ボクは直前までiPhone付属のEarPodsを使用していたのですが、聞いてみてあまりの音質の良さに驚いてしまいました。
操作性などもイヤホンの平らな部分をタップすることで再生や停止、次の曲へ変更などもできるので、かなり簡単に覚えられました。
さらに、AirPods Proにもある外音取り込み機能があります。外で聞きながら歩いても安心です。
本当にいい買い物をしたという気分でした。今でも毎日使っています。


以上が商品の紹介とボクが使用した感想でした。あなたも欲しくなってしまったのではないでしょうか。
今アマゾンでは、ボクが購入した時と近い価格の3万4000円で売られています。
ですがボクが確認したところ、アメリカの amazon.com では、
なんと1万円以上も安く売られていました。日本への送料を含めても、日本のアマゾンで買うよりも1万円は安くなっています。
これは買い時だと思います。AirPods Proが買えない今、同等のスペックを持つSennheiser Momentum True Wirelessがさらに安く買えるチャンスです。

この機会を逃す手はないのではないでしょうか。
